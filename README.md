# Halloween-Dashboard
This Dashboard explains the evaluation of the individuals who participate in Halloween 
    activities. It is used to understand the event planning. The following information we can get 
    from this dashboard.
•	Find the precipitation level 
•	Forecast the future event
•	Show year by year variation 
•	Display visitor strength
# Description
This Dashboard explains the evaluation of customers who are associated with the bank in the United Kingdom. It is used to understand financial performance, customer behavior, and overall banking operations. The following information we can get from this dashboard:
• Find the total deposits and loan distribution
• Forecast future financial growth and customer trends
• Show year-by-year variation in revenue and transactions
• Display total customer strength and account activity
